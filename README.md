# ❄️ Opensoure_MentalCare_chatbot 
숙명여자대학교 오픈소스프로그래밍 과목에서 구현한 '눈송이 멘탈케어 챗봇' 소스코드입니다. 저희 팀은 학교의 상징인 눈송이를 모티브로 눈송이 사용자들의 정신적인 부분을 케어해주는 전담 챗봇을 구현하였습니다. 

### ⏱️ 개발기간
2024.05.15 ~ 2024.06.21

## 🩵 기능 소개
1. 대화기능(챗봇)
2. 감정분석기능
3. 대화 초기화 기능
4. 응답시간 측정 기능
5. 대화 요약(로그) 기능
6. 로그 pdf파일로 저장 기능
7. 음악 재생 기능
8. 심리 관련 사이트 답변 기능
9. 우울증검사 기능
10. 스트레스검사 기능
